About me

항상 성실하게 성장하는 강지윤입니다.

프론트엔드 개발자가 되기 위해 리액트를 공부하는 중입니다. 프로젝트 경험을 쌓기 위해 프로젝트 캠프에 참여하였습니다. 기초가 탄탄하고, 경험이 많은 개발자가 되기 위해 꾸준히 공부할 계획입니다. 최근 들어서는 "타입스크립트"에 관심이 생겼습니다.

  공부 이외의 시간에는 보통 카페를 가서 커피를 마시거나, 독서를 즐겨야합니다. 야구 관람도 좋아하는 취미이지만 스트레스를 많이 받아 즐겨보지는 않습니다. 아침에 하는 러닝을 좋아하는데 더 날이 더워지기 전에 열심히 달릴 계획입니다. :)

Study Plan

1. HTML/CSS

시맨틱 태그를 활용하여 시맨틱한 마크업 구조를 능숙하게 짜고, flex&grid 연습을 많이 할 것이다. 웹 접근성을 높이는 HTML을 작성할 것이다.

2. JAVASCRIPT

주요 메서드를 능숙하게 사용하고, 콜백함수나 클로저 등 코어 자바스크립트의 개념을 확실히 정리할 것이다.

3. REACT

적절한 상황에 React Hooks를 올바르게 사용할 수 있도록 공부할 것이고, 컴포넌트별 작업에 능숙해질 것이다. 프로젝트 때 리액트 활용능력을 많이 키우고 싶다.
Daily

수업 때 배운 내용을 복습하며 직접 실습을 통해 연습할 것이다.

Weekly

한 주의 실습 과제들을 복습하고, 배운 내용을 활용해 새롭게 실습하 것이다.

Monthly

프로젝트 진행 방향에 대해 고민하고, 팀원들과 운영방식을 정해나갈 것이다. 프로젝트 전 기본기를 많이 쌓아놓을 것이다.
